# Release Notes for 1.0.8

## 简介

涂图流处理（特效相机） SDK 是涂图基于特效渲染引擎与采集相结合的 SDK 产品。产品分为 2 个分类，一种是使用完整的涂图相机 + 特效渲染引擎；一种是使用第三方采集 + 涂图特效渲染引擎。 前者更多是适用于自采集和视频录制场景，后者更多是适用于使用第三方直播 SDK 场景使用。

目前该 SDK 提供多项特效包括但不限于：

- 智能美肤；
- AI 微整形特效；
- AI 动态贴纸特效；
- AI 哈哈镜特效；
- AI 美妆
- 滤镜特效；
- 漫画滤镜；
- 变声特效；
